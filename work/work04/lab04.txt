[Soal nomor 1]
a. Perbedaan terlihat di dalam addressnya, address yang berada di local-char lebih panjang daripada global-char.
b. Karena variabel lokal hanya aktif saat fungsi tersebut dijalankan, sedangkan variabel global harus terus aktif.
c. Global and Local Variable Adress 
   - variabel global adalah variabel yang dapat digunakan oleh method mana saja karena tidak berada didalam sebuah method atau dideklarasikan diluar method.

   - variabel lokal adalah variabel yang hanya bisa digunakan oleh method tertentu karena variabelnya dideklarasikan didalam method tersebut.

   Sumber: https://www.tutorialspoint.com/cprogramming/c_scope_rules.htm

[Soal nomor 2]
a. Flags adalah parameter yang menjelaskan kondisi file tersebut.
b. Modes adalah parameter yang menjelaskan permission file tersebut.
c. Open dan Close I/O Process
   - Open berarti meminta sistem operasi untuk menggunakan sebuah file, jalur argument akan meminta secara spesifik file apa yang ingin dibuka serta ada flags dan modes yang akan menentukan bagaimana kita ingin menggunakan/mengeksekusi file tersebut.

   - Close berarti meminta sistem operasi untuk menghentikan segala proses yang terjadi dengan file descriptornya. Dan sistem operasi dapat menggunakan kembali file descriptor tersebut ketika dilakukan perintah Open.

   Sumber: http://web.eecs.utk.edu/~huangj/cs360/360/notes/Syscall-Intro/lecture.html
